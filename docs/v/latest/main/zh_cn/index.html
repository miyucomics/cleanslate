      凋零石板

      法术环垂直方向上的作用范围无法超出其石板的位置。虽然可以用导向石把媒质流导向上方，但不太美观。有传言称，媒质流在经过某种与信标有关的花后，法术环的影响范围即会向上方和下方扩展到无限远。

      平板型法术环

      平整的石板回路中总是会有促动石凸出来，真是相当引人不适。这种为人所不齿的差错绝不可容忍。让铁砧坠落到促动石上，就可把促动石拍到和一般的石板一样扁。其中的村民大概不会受损。把此类促动石放到漏斗或发射器上即可为其充能。


      还可以直接把石板从地板铺到墙上，再铺到天花板上，最后铺回来！同一格里也能放下多个石板；可借此随意改变媒质流的方向，不过这种“多重石板”上不可刻入图案。如果要让媒质流转过凸角，可以在石板下方放置空白促动石，也可让媒质流直接进入促动石。
